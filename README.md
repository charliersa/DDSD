# 繪圖軟體資訊整合 · Drawing & Design Software Directory

收錄線上與電腦版的繪圖、3D、設計與遊戲開發工具的目錄網站，可依分類、平台、價格篩選，查看介紹、連結官網，或加入比較。

## 內容

- **`react-app/`** — Vite + React 版本（主要版本）。詳見 [react-app/README.md](react-app/README.md)。
- **`繪圖軟體導覽.dc.html`** — 原始單檔 DC 元件版本。
- **`icon/`** — 軟體 Logo 圖示。
- **`image-slot.js` / `support.js`** — DC 版本所需的元件與執行環境。

## 啟動 React 版

```bash
cd react-app
npm install
npm run dev
```

瀏覽器開啟 http://localhost:5173/ 即可預覽。
